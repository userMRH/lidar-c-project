# LiDAR Veri İşleme Projesi

Bu proje, TOML formatındaki LiDAR sensör verilerini okuyup mesafe ölçümlerini işleyen C++ tabanlı bir uygulamadır.

## Özellikler

- TOML dosyası okuma
- Mesafe verilerini işleme
- Grafiksel çıktı oluşturma (SVG)
- Visual Studio proje dosyaları dahil

## Kullanılan Teknolojiler

- C++
- Visual Studio
- Dosya İşlemleri
- Veri Analizi

## Proje Amacı

LiDAR sensöründen alınan ham verileri işleyerek anlamlı çıktılar üretmek ve veri işleme mantığını uygulamalı olarak geliştirmektir.
